# Hapag
An app to help home cooks to find recipes using the ingredients that they have on hand.

## Getting Started
[Click Here](https://mydeveloperjourney.github.io/NYPD-Complaints-App/) to get started

## Technologies Used
- HTML
- CSS
- JavaScript/jQuery
- Google Fonts
- Spoonacular Recipe API
